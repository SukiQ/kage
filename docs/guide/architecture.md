# 架构分析

让 AI 阅读整个项目代码，自动生成模块依赖架构图与开发提示。

## 功能

- **AI 架构图**：识别模块（节点）、层级、职责与依赖关系（边），可视化呈现。
- **架构摘要**：AI 给出的整体架构说明。
- **AI 开发提示**：针对架构的改进与开发建议。
- **过程可见**：实时显示 AI 当前在阅读哪个文件 / 执行什么工具。
- **导出报告**：一键导出 Markdown 架构报告（含模块表、依赖关系、开发提示）。

## 使用步骤

1. 进入「架构分析」。
2. 点「AI 分析架构」，等待 AI 阅读代码（期间可看到实时进度）。
3. 查看生成的架构图、摘要与提示。
4. 需要时点导出按钮保存报告。
