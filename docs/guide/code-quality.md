# 代码质量

基于 SonarQube 扫描结果，量化展示代码缺陷，并提供 AI 辅助的修复闭环。

## 功能

- **指标概览**：Bugs、代码异味、重复率、技术债、可靠性、可维护性等卡片一目了然。
- **严重度分布**：BLOCKER / CRITICAL / MAJOR / MINOR / INFO 的可视化占比。
- **问题清单**：按严重度排列，显示位置（文件:行）与规则，支持修复 / 忽略操作。
- **单条 AI 修复**：针对单个问题发起修复，可填写「修复附言」；附言按规则（rule）全局记忆，下次同类问题自动复用。
- **一键批量修复**：一次性把所有待处理（open）问题交给 AI 修复。
- **Issue 生命周期**：修复 / 忽略 / 重新计入，状态按项目持久化。

## 使用步骤

1. 先在「总览」页执行一次扫描。
2. 进入「代码质量」，查看指标与问题清单。
3. 对单条问题点「修复」（可附言），或点「一键修复」批量处理。
4. 对无需处理的问题点「忽略」，需要时再「重新计入」。

::: tip 修复方式
AI 以 bypass 权限运行，会直接修改项目源码。可在右侧面板实时查看 AI 的思考与工具调用过程。
:::
