# 安全审查

聚焦漏洞与安全热点，结合 AI 给出处置建议，支持单条 / 一键修复与安全报告导出。

## 功能

- **安全指标**：漏洞数、安全热点数、安全评级（A–E）。
- **问题列表**：漏洞（vulnerability）与安全热点（security hotspot）分类展示，标注严重度与位置（文件:行 · 规则）。
- **单条修复**：对任意待处理漏洞/热点，点「修复」弹窗填写附言（按错误码自动记忆），由 AI 直接改源码完成安全加固，修复完成自动标记为「已修复」。
- **一键修复**：一键让 AI 批量修复所有待处理安全问题，相同规则用一致的安全修复方式。
- **忽略 / 重新计入**：可忽略误报或暂不处理的问题，后续随时重新计入。
- **导出安全报告**：一键导出 Markdown 安全报告（安全概览 + 严重度分布 + 问题清单，含每条状态与附言）。

## 使用步骤

1. 先在「总览」页执行扫描。
2. 进入「安全审查」，查看漏洞与热点列表。
3. **修复**：
   - 单条：在某条问题上点「修复」，填写修复附言后由 AI 改码加固。
   - 批量：点工具栏「一键修复」，AI 逐一修复所有待处理安全问题。
   - 过程与结果展示在右侧 AI 面板；修复完成自动标记「已修复」并按项目持久化。
4. 必要时点「忽略」暂缓某问题，或对已忽略问题点「重新计入」。
5. 点工具栏下载图标「导出安全报告」，保存为 Markdown。

::: tip 语言无关
安全修复同样委托 Claude Code 直接改源码，不按语言硬编码，Java / Python / Dart / 前端等均可。
:::
